# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-XX

### Added
- Initial release of Roommate CLI
- Real-time streaming chat interface with AI
- Beautiful terminal UI using Rich library
- Support for both public and private API servers
- Authentication via configuration file
- Live response streaming with spinner animation
- Typewriter effect for final responses
- Multiple exit command options (exit, quit, :q, bye, sair)
- Keyboard interrupt handling
- Server connectivity check
- Markdown rendering for responses
- Colorful panels and formatted output

### Features
- Connect to public API at https://roommate.theushen.me
- Optional unlimited access with API password
- Interactive chat experience
- Real-time message streaming
- Professional terminal interface
